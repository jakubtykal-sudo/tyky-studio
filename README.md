# Tyky Studio

Repozitář projektů a marketingových podkladů pro **Klimahome Group**.

Tyky Studio je interní studio Jakuba Tykala, které se stará o značku, obchodní materiály a digitální podporu firem ve skupině.

---

## 🏢 Firmy ve skupině

- **Klimahome** — vlajková HVAC firma. Klimatizace, tepelná čerpadla, montáže, servis.
- **Klimadoma / HST Group** — technologicko-servisní zázemí (klimatizace, TČ, B2B servis, outsourcing techniků).
- **Tyky Studio** — značka, marketing, obsah, designové podklady.
